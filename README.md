# Student Management System (C++)

![C++](https://img.shields.io/badge/Language-C%2B%2B-blue)
![License](https://img.shields.io/badge/License-MIT-green)

A **console-based Student Management System** implemented in C++ for data structure practice. This program allows you to **add, view, search, edit, and delete student records** stored in a text file.

---

## Features

- Add new student records with the following details:
  - Roll Number
  - Name
  - Father's Name
  - Address
  - Blood Group
  - Donor Status
  - Phone Number
  - T-Shirt Size
  - Active Status
- View all student records in a **tabular format**, sorted by roll number.
- Search for a student record by roll number.
- Edit existing student records.
- Delete student records.
- Persistent storage using a text file (`stuData.txt`).

---

## Technologies Used

- **C++**: File I/O, Structs, Vectors, Sorting
- **Console-based interface**
